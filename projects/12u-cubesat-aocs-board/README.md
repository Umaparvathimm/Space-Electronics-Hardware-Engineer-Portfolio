## Overview
This project involves the **hardware development of an Attitude and Orbit Control System (AOCS) board** for a **12U CubeSat platform**, supporting precise attitude determination, control, and safe-mode operation for mission-critical spacecraft functions.

The AOCS electronics are designed to interface with a diverse set of **sensors and actuators**, enabling attitude estimation, closed-loop control, and fault-aware operation within strict **SWaP (Size, Weight, and Power)** constraints. The board forms a central element of the spacecraftâ€™s control architecture and is designed for reliable operation in a space environment.

The work focuses on translating **system-level AOCS requirements** into robust, testable hardware, with careful consideration of power integrity, signal interfacing, fault containment, and subsystem integration.
